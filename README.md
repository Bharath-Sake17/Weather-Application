# 🌦️ Weather Application

A simple and responsive **Weather Application** that provides real-time weather information for any city using a weather API. This project demonstrates API integration, asynchronous JavaScript, and dynamic UI updates.

---

## 🚀 Features

- 🌍 Search weather by city name  
- 🌡️ Displays current temperature  
- ☁️ Shows weather condition (Cloudy, Sunny, Rainy, etc.)  
- 💧 Humidity and wind speed details  
- 🔄 Real-time data using API  
- ⚡ Fast and responsive UI  
- 📱 Mobile-friendly design  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **API:** OpenWeatherMap API (or similar)  
- **Concepts Used:**
  - Fetch API  
  - Async / Await  
  - Promises  
  - DOM Manipulation

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/Bharath-Sake17/Weather-Application.git
```

2. Navigate to project folder:
```bash
cd Weather-Application
```

3. Open `index.html` in browser  
   OR use Live Server in VS Code

---

## 🔑 API Configuration

1. Get API key from:
👉 https://openweathermap.org/api  

2. Replace API key in `script.js`:
```javascript
const apiKey = "YOUR_API_KEY";
```


---

## 📌 How It Works

1. User enters a city name  
2. App sends request using **Fetch API**  
3. API returns weather data in JSON format  
4. Data is parsed and displayed on UI
